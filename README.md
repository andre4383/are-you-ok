# are you ok?

A minimalist website with random positive messages.

## Features

Random positive messages on each click
Cormoant Garamond serif typography
GSAP-powered animations
Minimalist black and white design
Fully responsive
Built with TypeScript

## How It Works

Click "are you ok?" to display a random positive message
Click the message to return to the home screen
Click again to see a new different message

## Technologies

Next.js 14+ (App Router)
TypeScript
Tailwind CSS
GSAP
Google Fonts (Cormorant Garamond)
